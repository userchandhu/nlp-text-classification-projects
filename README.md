# NLP Text Classification Projects

## 📌 Overview
This repository contains NLP projects including spam detection and sentiment analysis.

## 🛠️ Technologies Used
- Python
- NLTK
- Scikit-learn

## ⚙️ Approach
- Text preprocessing (tokenization, stopwords, stemming, lemmatization)
- Feature extraction using TF-IDF and Bag of Words
- Model building using Multinomial Naive Bayes

## 📊 Projects Included
- Email Spam Detection
- Sentiment Analysis (IMDB dataset)

## 🧠 Key Learning
Text preprocessing and feature engineering are critical in NLP tasks.
